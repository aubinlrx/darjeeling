Darjeeling
==========

Darjeeling is a nice and simple full-stack framework to easily create web application on [NodeJS](http://nodejs.org). He is based on [ExpressJS](http://expressjs.com/) and some nice modules.

## Dependencies
- Async **3.1.0**
- BodyParser **1.19.0**
- Express **4.17.1**
- Lodash **4.17.15**
- Moment **2.24.0**
- Mongoose **5.8.9**
- NodeTime **0.8.17**
- Pug **2.0.4**
- ServeFavicon **2.5.0**
- ServeStatic **1.14.1**
- Winston **3.2.1**

## Install dependencies
**With [node](http://nodejs.org)**

``` sh
$ yarn install
```

## Start server
``` sh
$ yarn start
```
